業務 SOP 教育訓練系統 (Sales SOP Training Portal)

這是一個基於 Web 的互動式教育訓練平台，旨在幫助業務同仁快速掌握標準作業流程 (SOP)。本系統將傳統的 Word 文件轉化為易於閱讀、互動性強的網頁應用程式，並支援跨國團隊的雙語需求與手機行動學習。

🚀 功能特色 (Features)

單一檔案架構 (Single HTML File)：無需安裝任何伺服器或編譯工具，下載即可直接在瀏覽器運行。

雙語切換系統 (Bilingual Support)：內建「繁體中文 / 越南文」一鍵切換，適合跨國工廠與業務團隊使用。

智慧進度記憶 (Smart Progress Tracking)：

自動記錄閱讀章節，下次開啟自動跳轉。

測驗滿分後，章節列表會顯示綠色勾勾 (✅) 標記成就。

支援一鍵重置學習進度。

行動裝置優化 (Mobile Optimized)：

針對手機觸控優化操作體驗。

響應式側邊導覽列，點擊後自動收合。

互動式學習 (Interactive Content)：

流程步驟：卡片式呈現，清晰易讀。

時程與職責：整合「標準時程表」與「職責矩陣」，明確定義各角色權責與作業時間。

隨堂測驗：每章節附帶互動測驗與即時評分功能。

🛠️ 如何使用 (How to Use)

下載：將專案中的 index.html (或 SOP_Training_System.html) 下載到您的電腦。

開啟：直接雙擊檔案，使用 Chrome、Edge 或 Safari 瀏覽器開啟即可。

部署 (推薦)：

將檔案上傳至 GitHub Repository。

開啟 GitHub Pages 功能 (Settings -> Pages -> Branch: main -> Save)。

分享生成的網址給團隊成員，即可在手機或電腦上隨時學習。
